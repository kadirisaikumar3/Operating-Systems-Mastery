# MCQs

### Q1

Smallest unit of execution is

A. Process

B. Thread

C. Kernel

D. Program

✅ Answer: **B**

---

### Q2

Threads inside a process share

A. Stack

B. Registers

C. Heap

D. Program Counter

✅ Answer: **C**

---

### Q3

Which thread handles user interaction?

A. UI Thread

B. Network Thread

C. Audio Thread

D. GPU Thread

✅ Answer: **A**
