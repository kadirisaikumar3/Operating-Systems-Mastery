# 📘 Chapter 4 - Threads

## 📚 Overview

A Thread is the smallest unit of execution within a process. While a process represents a running program, threads allow multiple tasks to execute concurrently inside the same process by sharing resources such as memory and files.

Modern applications like Google Chrome, Microsoft Word, Visual Studio Code, and Android apps heavily rely on multithreading to improve responsiveness and performance.

This chapter covers Thread concepts, Thread Life Cycle, Types of Threads, Thread Control Block (TCB), Multithreading Models, User-Level vs Kernel-Level Threads, Thread Scheduling, and real-world applications.
