# 📘 Chapter 15 - Inter Process Communication (IPC)

## 📚 Overview

Inter Process Communication (IPC) enables processes to exchange data and synchronize their activities.

Since processes have separate memory spaces, IPC mechanisms provide safe and efficient communication.

This chapter covers Shared Memory, Message Passing, Pipes, Named Pipes, Sockets, Signals, Semaphores, Producer Consumer Problem, and interview-focused concepts.
