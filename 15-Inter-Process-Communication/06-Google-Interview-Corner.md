# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

IPC is a very common Operating Systems interview topic.

---

# Question 1

What is IPC?

Expected Answer

IPC is a mechanism that allows processes to exchange data and synchronize their actions.

---

# Question 2

Shared Memory vs Message Passing

| Shared Memory | Message Passing |
|--------------|----------------|
| Faster | Slower |
| Requires Synchronization | Easier |
| Same Machine | Same/Remote |

---

# Question 3

What is a Pipe?

Expected Answer

A Pipe is a communication channel used between related processes.

---

# Question 4

What is a Socket?

Expected Answer

A Socket is an endpoint used for network communication.

---

# Question 5

What is a Semaphore?

Expected Answer

A synchronization mechanism used to prevent race conditions.
