# 10-Common-Mistakes
# ❌ Common Mistakes

## Mistake 1

Thinking a File and File System are the same.

File

A collection of data.

File System

Manages all files.

---

## Mistake 2

Confusing Hard Link with Soft Link.

Hard Link

Points to inode.

Soft Link

Points to pathname.

---

## Mistake 3

Thinking FAT supports Journaling.

Wrong.

Only NTFS, ext4, APFS support Journaling.

---

## Mistake 4

Ignoring File Permissions.

Permissions are essential for security.

---

## Mistake 5

Confusing Allocation Methods.

Always compare

Contiguous

↓

Linked

↓

Indexed

---

## Interview Advice

Explain

File

↓

Directory

↓

Allocation

↓

Protection

↓

Journaling
