# 03-Real-World-Example
# 🌍 Real-World Examples

File Systems are used by every operating system to organize, store, retrieve, and protect data.

---

# Example 1: Windows (NTFS)

Windows primarily uses the NTFS (New Technology File System).

Features

- Journaling
- File Permissions
- Compression
- Encryption (EFS)
- Large File Support

Real Example

When you save a Microsoft Word document, NTFS stores the file metadata, permissions, timestamps, and disk location.

---

# Example 2: Linux (ext4)

Linux distributions commonly use ext4.

Features

- Journaling
- Fast Recovery
- Extents
- Inodes
- Large File Support

Example

Ubuntu stores user files inside:

/home/username/

---

# Example 3: Android

Android uses ext4 or F2FS depending on the device.

Responsibilities

- App storage
- Media storage
- Internal memory management
- File permissions

---

# Example 4: macOS (APFS)

Apple uses APFS.

Features

- Snapshots
- Encryption
- Space Sharing
- Copy-on-Write

---

# Example 5: Cloud Storage

Services like Google Drive and OneDrive use distributed file systems to store data across multiple servers.

Benefits

- High Availability
- Fault Tolerance
- Scalability

---

# Example 6: Database Systems

Databases use file systems to store

- Tables
- Indexes
- Logs
- Backup Files
