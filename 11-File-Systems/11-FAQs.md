# 11-FAQs
# ❓ Frequently Asked Questions

## What is a File System?

A mechanism that organizes and manages files on storage devices.

---

## Which file system does Windows use?

NTFS.

---

## Which file system does Linux use?

ext4.

---

## What is an Inode?

A data structure storing metadata about a file.

---

## What is Journaling?

Recording file system changes before writing them to disk for recovery.

---

## Hard Link vs Soft Link?

Hard Link → Inode

Soft Link → File Path

---

## Which allocation method is fastest?

Contiguous Allocation.

---

## Which allocation method avoids External Fragmentation?

Linked Allocation.

---

## Where are File Systems used?

- Windows
- Linux
- Android
- macOS
