## Q1

Which file system is commonly used in Windows?

A. ext4

B. FAT

C. NTFS

D. APFS

NTFS

## Q2

Which Linux file system is most common?

A. FAT32

B. ext4

C. NTFS

D. exFAT

✅ Answer

ext4

## Q3

Which allocation method provides fastest access?

A. Linked

B. Indexed

C. Contiguous
