# 📂 File System Concepts

## What is a File System?

A File System is a mechanism used by the Operating System to organize and manage files on secondary storage devices.

It defines:

- How files are stored
- How files are retrieved
- How files are organized
- How files are protected

---

## Definition

> A File System is a collection of methods and data structures that the Operating System uses to control how data is stored and retrieved.

---

## Why is a File System Needed?

Without a File System:

- Files cannot be located easily.
- Storage becomes unorganized.
- Data retrieval becomes slow.
- Security becomes difficult.

---

## Responsibilities

- Create files
- Delete files
- Read files
- Write files
- Rename files
- Organize directories
- Allocate disk space
- Protect files

---

## Advantages

- Organized Storage
- Fast Access

Your laptop stores:

- Photos
