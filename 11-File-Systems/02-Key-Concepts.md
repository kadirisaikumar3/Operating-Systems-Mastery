# 02-Key-Concepts
# 🔑 Key Concepts

## File

A File is a named collection of related information stored on secondary storage.

Examples

- notes.txt
- image.png
- video.mp4
- report.pdf

---

## File Attributes

Each file contains metadata.

Examples

- Name
- Size
- Owner
- Creation Date
- Last Modified
- Permissions
- Location

---

## File Operations

Operating Systems support:

- Create
- Open
- Read
- Write
- Close
- Rename
- Delete

---

## File Access Methods

### Sequential Access

Data is read from beginning to end.

Example

Watching a movie.

---

### Direct Access

Any location can be accessed directly.

Example

Opening page 150 in a PDF.

---

### Indexed Access

Uses an index for faster retrieval.

Example

Database systems.

---

## Directory

A directory stores files and subdirectories.

Examples

Documents

Downloads

Pictures

Music

---

- Two-Level
