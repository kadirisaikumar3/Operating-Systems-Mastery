# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

Memory Management is one of Google's favorite Operating System topics.

---

# Question 1

What is Memory Management?

Expected Answer

Memory Management allocates, protects, and optimizes RAM usage for processes.

---

# Question 2

Logical vs Physical Address

| Logical Address | Physical Address |
|-----------------|------------------|
| Generated by CPU | Generated by MMU |
| Virtual Address | Actual RAM Address |

---

# Question 3

Explain MMU.

Expected Answer

The Memory Management Unit translates Logical Addresses into Physical Addresses.

---

# Question 4

Internal vs External Fragmentation

| Internal | External |
|-----------|----------|
| Inside allocated block | Between free blocks |
| Wasted allocated memory | Non-contiguous free memory |

---

# Question 5

Which allocation algorithm is fastest?

Expected Answer

First Fit

---

# Question 6

Which allocation algorithm minimizes wasted space?

Expected Answer

Best Fit
