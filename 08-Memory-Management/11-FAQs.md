# 11-FAQs
# ❓ Frequently Asked Questions

## What is Memory Management?

Managing RAM efficiently for multiple processes.

---

## What is MMU?

Memory Management Unit.

It converts Logical Addresses into Physical Addresses.

---

## What is Swapping?

Moving processes between RAM and disk.

---

## What is Internal Fragmentation?

Unused space inside allocated memory.

---

## What is External Fragmentation?

Free memory exists but is not contiguous.

---

## What is Compaction?

Combining free memory blocks into one large block.

---

## Which algorithm is fastest?

First Fit.

---

## Which algorithm minimizes wasted memory?

Best Fit.

---
