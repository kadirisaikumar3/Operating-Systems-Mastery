# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

I/O Systems are frequently discussed in Operating System interviews.

---

# Question 1

What is an I/O System?

Expected Answer

An I/O System manages communication between CPU, memory, and external devices.

---

# Question 2

Programmed I/O vs Interrupt I/O

| Programmed I/O | Interrupt I/O |
|---------------|---------------|
| CPU Polling | Device Interrupt |
| Wastes CPU | Better Utilization |
| Simple | More Efficient |

---

# Question 3

What is DMA?

Expected Answer

DMA allows devices to transfer data directly to memory without CPU involvement.

---

# Question 4

Why is DMA faster?

Expected Answer

Because CPU is not involved in every byte transfer.

---

# Question 5

What is a Device Driver?

Expected Answer

Software that enables the Operating System to communicate with hardware devices.
