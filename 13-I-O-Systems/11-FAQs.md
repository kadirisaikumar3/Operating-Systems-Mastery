# 11-FAQs
# ❓ Frequently Asked Questions

## What is an I/O System?

A subsystem that manages communication between computer components and external devices.

---

## What is DMA?

Direct Memory Access allows direct transfer between device and memory.

---

## Why is DMA faster?

Because CPU does not manage every transfer.

---

## What is Interrupt I/O?

Device interrupts CPU when service is needed.

---

## What is Buffering?

Temporary storage used during data transfer.

---

## What is Spooling?

Managing device requests through a queue.

---

## What is a Device Driver?

Software that enables communication with hardware.

---

## Where are I/O Systems used?

- Operating Systems
- Databases
- Cloud Systems
- Networking
- Mobile Devices
