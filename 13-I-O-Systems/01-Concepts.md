# 🖥️ I/O Systems Concepts

## What is an I/O System?

An I/O System manages communication between the CPU, memory, and external devices.

Examples

- Keyboard
- Mouse
- Printer
- Monitor
- Disk
- Network Card

---

## Why is I/O Needed?

Without I/O:

- Users cannot interact with computers.
- Data cannot be stored.
- Programs cannot receive input.
