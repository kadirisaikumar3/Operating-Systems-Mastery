# MCQs

## Q1

Which method wastes CPU time?

A. DMA

B. Interrupt I/O

C. Programmed I/O

D. SSD

✅ Answer

Programmed I/O

---

## Q2

Which method uses direct transfer between device and memory?

A. DMA

B. Interrupt I/O

C. FCFS

D. RAID

✅ Answer

DMA

---

## Q3

Which software controls hardware devices?

A. Kernel

B. Device Driver

C. Compiler

D. Scheduler

✅ Answer

Device Driver
