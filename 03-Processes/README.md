# 📘 Chapter 3 - Processes

## 📚 Overview

A Process is a program in execution. It is one of the most fundamental concepts in Operating Systems because every application running on a computer is executed as one or more processes.

This chapter explains how processes are created, executed, scheduled, synchronized, and terminated. It also introduces Process Control Block (PCB), Process States, Context Switching, and Process Scheduling Basics.

Understanding Processes is essential for learning Threads, CPU Scheduling, Deadlocks, Synchronization, and Memory Management.
