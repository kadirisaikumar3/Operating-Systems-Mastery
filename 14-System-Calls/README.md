# 📘 Chapter 14 - System Calls

## 📚 Overview

System Calls provide an interface between user programs and the Operating System.

Applications cannot directly access hardware resources. Instead, they request services from the kernel through system calls.

This chapter covers System Calls, User Mode vs Kernel Mode, Types of System Calls, System Call Execution Flow, Linux System Calls, and interview-focused concepts.
