## Q1

Which mode has full privileges?

A. User Mode

B. Guest Mode

C. Kernel Mode

D. Safe Mode

✅ Answer

Kernel Mode

## Q2

Which system call creates a process?

A. open()

B. fork()

C. read()

D. close()

✅ Answer

fork()

## Q3

Which system call opens a file?

A. wait()

B. exec()

C. open()
