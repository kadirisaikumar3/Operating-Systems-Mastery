# 02-Key-Concepts
# 🔑 Key Concepts

## Process Control Calls

Used for process management.

Examples

- fork()
- exec()
- wait()
- exit()

---

## File Management Calls

Used for file operations.

Examples

- open()
- read()
- write()
- close()

---

## Device Management Calls

Used for device access.

Examples

- ioctl()
- read()
- write()

---

## Information Maintenance Calls

Used to retrieve system information.

Examples

- getpid()
- time()
- uname()

---

## Communication Calls

Used for process communication.

Examples

- pipe()
- socket()
- send()
- recv()

---

## Protection Calls

Used for permissions and security.

Examples

- chmod()
- access()

---

## Frequently Asked

System Calls act as a bridge between user applications and the Operating System.
