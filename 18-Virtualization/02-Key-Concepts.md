# 02-Key-Concepts
# 🔑 Key Concepts

## Type 1 Hypervisor

Runs directly on hardware.

Examples

- VMware ESXi
- Hyper-V
- Xen

Advantages

- High Performance
- Better Security

---

## Type 2 Hypervisor

Runs on top of an OS.

Examples

- VirtualBox
- VMware Workstation

Advantages

- Easy Setup

---

## Virtual Machines

Each VM has:

- Guest OS
- CPU
- Memory
- Storage

---

## Containers

Lightweight virtualization.

Examples

- Docker
- Kubernetes

---

## VM vs Container

VM

- Own OS
- More Isolation

Container

- Shares Host Kernel
- Faster
