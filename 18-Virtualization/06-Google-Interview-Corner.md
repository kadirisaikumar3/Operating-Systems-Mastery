# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

Virtualization is important for Cloud and Infrastructure interviews.

---

# Question 1

What is Virtualization?

Expected Answer

Virtualization creates virtual versions of computing resources.

---

# Question 2

What is a Hypervisor?

Expected Answer

A software layer that creates and manages Virtual Machines.

---

# Question 3

Type 1 vs Type 2 Hypervisor

| Type 1 | Type 2 |
|----------|---------|
| Direct Hardware Access | Runs on Host OS |
| Faster | Easier Setup |
| Enterprise Usage | Development Usage |

---

# Question 4

VM vs Container

| VM | Container |
|-----|-----------|
| Own OS | Shared Kernel |
| Heavy | Lightweight |
| More Isolation | Faster Startup |

---

# Question 5

Docker vs Kubernetes

Expected Answer

Docker creates containers.

Kubernetes manages containers.
