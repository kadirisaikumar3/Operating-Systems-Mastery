# 10-Common-Mistakes
# ❌ Common Mistakes

## Mistake 1

Thinking Virtualization and Containers are identical.

They are different technologies.

---

## Mistake 2

Confusing Type 1 and Type 2 Hypervisors.

Type 1

Runs on Hardware

Type 2

Runs on OS

---

## Mistake 3

Ignoring Resource Overhead.

Virtual Machines consume more resources.

---

## Mistake 4

Thinking Docker is a Hypervisor.

Docker is a container platform.

---

## Mistake 5

Confusing Docker and Kubernetes.

Docker

Creates Containers

Kubernetes

Manages Containers

---

Hypervisor

VM
