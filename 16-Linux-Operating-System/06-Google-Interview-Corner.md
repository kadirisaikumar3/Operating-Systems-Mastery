# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

Linux is extremely important for Software Engineers.

---

# Question 1

What is Linux?

Expected Answer

Linux is an open-source UNIX-like Operating System.

---

# Question 2

What is the Linux Kernel?

Expected Answer

The Kernel is the core component that manages processes, memory, devices, and files.

---

---

# Question 4

What are Linux Permissions?

Expected Answer

Permissions control file access.

r = Read

w = Write

x = Execute

---

# Question 5

Explain chmod 755.

Expected Answer

Owner = rwx

Group = r-x

Others = r-x
