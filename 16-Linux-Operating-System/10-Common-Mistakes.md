# ❌ Common Mistakes

## Mistake 1

Thinking Linux and UNIX are identical.

Linux is UNIX-like.

---

## Mistake 2

Confusing Kernel and Shell.

Kernel

Core OS Component

Shell

Command Interface

---

## Mistake 3

Using chmod incorrectly.

Always verify permissions before changing them.

---

## Mistake 4

Running commands as root unnecessarily.

Can damage the system.

---

## Mistake 5

Ignoring system logs.

Logs help diagnose problems.
