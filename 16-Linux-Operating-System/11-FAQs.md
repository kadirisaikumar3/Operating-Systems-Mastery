## What is Linux?

An open-source UNIX-like Operating System.

---

## What is the Linux Kernel?

The core component that manages resources.

---

## What is a Linux Distribution?

A Linux-based operating system package.

Examples:

- Ubuntu
- Fedora
- Debian

---

## What is chmod?

Command used to modify permissions.

---

## What is ps?

Displays running processes.

## What is top?

Real-time process monitoring tool.
