# 🐧 Linux Concepts

## What is Linux?

Linux is an open-source Operating System developed by Linus Torvalds in 1991.

---

## Features of Linux

- Open Source
- Multiuser
- Multitasking
- Portable
- Secure
- Stable
- High Performance

---

## What is the Linux Kernel?

The Kernel is the core component of Linux.

Responsibilities:

- Process Management
- Memory Management
- File Management
- Device Management
- Networking

---

## Linux Usage

- Servers
- Cloud Computing
- Android
- Embedded Systems
- Supercomputers

---

## Real World Example

Most cloud platforms use Linux servers because of stability and performance.

Examples:

- AWS

- Azure
