# 🚀 Google Interview Corner

Security concepts are commonly discussed in system and infrastructure interviews.

---

# Question 1

What is CIA Triad?

Expected Answer

Confidentiality

Integrity

Availability

---

# Question 2

Authentication vs Authorization

| Authentication | Authorization |
|---------------|---------------|
| Who are you? | What can you access? |
| Identity Verification | Permission Control |

---

# Question 3

What is RBAC?

Expected Answer

Role Based Access Control grants permissions based on user roles.

---

# Question 4

What is Encryption?

Expected Answer

Converting readable data into unreadable form to protect information.

---

# Question 5

What is a Firewall?

Expected Answer

A security system that filters incoming and outgoing network traffic.
