# MCQs

## Q1

Which problem occurs without synchronization?

A. Starvation

B. Race Condition

C. Paging

D. Fragmentation

✅ Answer

## Q2

Which synchronization primitive allows only one thread?

A. Counting Semaphore

B. Binary Semaphore

C. Queue

Binary Semaphore

synchronized
