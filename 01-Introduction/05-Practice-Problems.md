# Multiple Choice Questions

### Q1

The Operating System acts as an interface between

A. CPU and RAM

B. User and Hardware

C. RAM and Disk

D. Keyboard and Monitor

✅ Answer: **B**

---

### Q2

Which is system software?

A. MS Word

B. Chrome

C. Windows

D. VS Code

✅ Answer: **C**

---

### Q3

Which component directly communicates with hardware?

A. Shell

B. Kernel

C. Browser

D. Compiler

✅ Answer: **B**

---

### Q4

Which of the following is NOT an Operating System?

A. Linux

B. Android

C. Windows

D. Python

✅ Answer: **D**
