# 06-Google-Interview-Corner
# 🚀 Google Interview Corner

This section contains conceptual questions commonly asked in Google, Microsoft, Amazon and other Product-Based Company interviews.

---

# Question 1

What is an Operating System?

Expected Answer:

An Operating System is system software that manages hardware resources and provides services for application software while acting as an interface between users and hardware.

---

# Question 2

Why can't applications directly access hardware?

Expected Points

- Security
- Resource Management
- Hardware Abstraction
- Device Independence

---

# Question 3

What happens internally when you open Google Chrome?

Expected Flow

User Click

↓

Operating System receives request

↓

Kernel creates Process

↓

Memory Allocation

↓

CPU Scheduling

↓

Program Execution

↓

Chrome Window Appears

---

# Question 4

Explain the difference between Program and Process.

| Program | Process |
|----------|----------|
| Stored on Disk | Running in Memory |
| Passive | Active |
| Static | Dynamic |

---

# Question 5

Kernel vs Shell

| Kernel | Shell |
|---------|--------|
| Core of OS | User Interface |
| Hardware Interaction | User Interaction |
| Runs in Kernel Mode | Runs in User Mode |

---

# Question 6

Why is Kernel called the Heart of an Operating System?

Expected Answer

Because every hardware request passes through the Kernel. It manages memory, CPU scheduling, device communication and process execution.

---

# Question 7

Can two Operating Systems run simultaneously?

Answer

Yes.

Using

- Virtual Machines
- Dual Boot
- Containers (OS-level virtualization)

---

# Question 8

Explain Multitasking.

Expected Points

- CPU Scheduling
- Context Switching
- Time Sharing

---

# Question 9

Why do Operating Systems require Scheduling?

Expected Points

- Fair CPU Allocation
- Better Performance
- Reduced Waiting Time
- Higher CPU Utilization

---

# Question 10

Real Interview Question

Suppose Windows suddenly disappears from your laptop.

Can Chrome still run?

Answer

No.

Chrome depends on Operating System services such as

- Memory Allocation
- File Access
- Device Drivers
- CPU Scheduling

Without an Operating System, application software cannot execute.
