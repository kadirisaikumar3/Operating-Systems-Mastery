# 10-Common-Mistakes
# ❌ Common Mistakes

## Mistake 1

Thinking Operating System is an Application Software.

Correct:

Operating System is **System Software**.

---

## Mistake 2

Confusing Program and Process.

Program

↓

Stored on Disk

Process

↓

Running in Memory

---

## Mistake 3

Kernel and Shell are same.

Wrong.

Kernel manages Hardware.

Shell communicates with User.

---

## Mistake 4

Thinking CPU executes multiple programs simultaneously.

Actually,

CPU rapidly switches between processes using Context Switching.

---

## Mistake 5

Thinking BIOS is Operating System.

Wrong.

BIOS loads the Operating System.

---

## Mistake 6

Thinking Android is not an Operating System.

Wrong.

Android is Linux-based Operating System.

---

## Interview Advice

Always give

Definition

↓

Working

↓

Example

↓

Advantages

instead of only definitions.
