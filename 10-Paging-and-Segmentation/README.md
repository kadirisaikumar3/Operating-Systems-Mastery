# 📘 Chapter 10 - Paging & Segmentation

## 📚 Overview

Paging and Segmentation are two important memory management techniques used by Operating Systems to efficiently allocate memory and execute programs.

Paging divides memory into fixed-size pages and frames, eliminating external fragmentation. Segmentation divides programs into logical units such as code, data, and stack, making memory organization more meaningful.

This chapter covers Paging, Segmentation, Address Translation, Page Tables, Segment Tables, Internal and External Fragmentation, Shared Memory, and Java Memory concepts.
