# 11-FAQs
# ❓ Frequently Asked Questions

## What is Paging?

A memory management technique that divides memory into fixed-size pages and frames.

---

## What is Segmentation?

A memory management technique that divides programs into logical units.

---

## What is a Page Table?

A data structure that maps logical pages to physical frames.

---

## What is a Segment Table?

A table containing Base Address and Limit for each segment.

---

## Which fragmentation occurs in Paging?

Internal Fragmentation.

---

## Which fragmentation occurs in Segmentation?

External Fragmentation.

---

## What is TLB?

Translation Lookaside Buffer.

A cache used for faster address translation.

---

## Where are Paging and Segmentation used?

- Windows
- Linux
- Android
- JVM
- Modern Operating Systems
