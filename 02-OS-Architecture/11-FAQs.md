# 11-FAQs
# ❓ Frequently Asked Questions

## What is OS Architecture?

The internal design of an Operating System.

---

## Why is Architecture important?

Because it determines performance, security and maintainability.

---

## Which Architecture does Linux use?

Monolithic Kernel.

---

## Which Architecture does Windows use?

Hybrid Kernel.

---

## Which Architecture does macOS use?

Hybrid (XNU).

---

## What is User Space?

Area where applications execute.

---

## What is Kernel Space?

Area where Operating System executes.

---

## What are System Calls?

Functions used by applications to request Operating System services.

---

## Why can't applications access hardware directly?

For security and resource management.

---

## What is Device Driver?

Software that enables communication between Operating System and Hardware.

---

## Which Architecture is fastest?

Generally,

Monolithic Kernel.

---

## Which Architecture is most secure?

Generally,

Microkernel.

---

## Which Architecture is used in Android?

Linux Kernel.

---

## Which Architecture is used by Google Servers?

Linux Monolithic Kernel.
