# Multiple Choice Questions

### Q1

Linux uses which architecture?

A. Hybrid

B. Microkernel

C. Monolithic

D. Layered

✅ Answer: **C**

---

### Q2

Windows uses

A. Microkernel

B. Hybrid

C. Exokernel

D. Nanokernel

✅ Answer: **B**

---

### Q3

Which architecture provides better fault isolation?

A. Monolithic

B. Microkernel

C. Hybrid

D. Layered

✅ Answer: **B**

---

### Q4

Applications normally execute in

A. Kernel Space

B. BIOS

C. User Space

D. Boot Loader

✅ Answer: **C**
