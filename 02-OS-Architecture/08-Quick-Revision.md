# ⚡ Quick Revision

## Operating System Architecture

- Defines the internal design of an Operating System.
- Determines how OS components interact with hardware.

---

## Types of Architectures

- Monolithic Kernel
- Microkernel
- Hybrid Kernel
- Layered Architecture
- Modular Kernel
- Exokernel
- Nanokernel

---

## Monolithic Kernel

✔ High Performance

✔ Fast Communication

❌ Large Kernel

Example:

- Linux
- UNIX

---

## Microkernel

✔ Secure

✔ Reliable

✔ Modular

❌ Slightly Slower

Examples:

- MINIX
- QNX

---

## Hybrid Kernel

Combination of

- Monolithic
- Microkernel

Examples

- Windows
- macOS

---

## User Space vs Kernel Space

| User Space | Kernel Space |
|------------|--------------|
| Applications | Operating System |
| Limited Access | Full Hardware Access |

---

## System Calls

Applications communicate with the Kernel using System Calls.

---

## Device Drivers

Help the Operating System communicate with hardware.

Examples

- Printer
- GPU
- Network Card
- Keyboard
