# 10-Common-Mistakes
# ❌ Common Mistakes

## Mistake 1

Thinking Kernel and Architecture are the same.

Correct:

Kernel is a component.

Architecture is the overall design.

---

## Mistake 2

Thinking Linux is Microkernel.

Wrong.

Linux uses Monolithic Kernel.

---

## Mistake 3

Thinking Windows is Monolithic.

Wrong.

Windows uses Hybrid Kernel.

---

## Mistake 4

Applications directly access Hardware.

Wrong.

Applications always use System Calls.

---

## Mistake 5

Microkernel is always better.

Wrong.

It depends on requirements.

Performance

↓

Monolithic

Security

↓

Microkernel

---

## Interview Advice

Always explain

Why

instead of

What.

Interviewers evaluate reasoning.
