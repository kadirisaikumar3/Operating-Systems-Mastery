# 📘 Chapter 12 - Disk Scheduling

## 📚 Overview

Disk Scheduling is a technique used by the Operating System to determine the order in which disk I/O requests are serviced.

Efficient disk scheduling improves performance by reducing seek time, rotational latency, and overall disk access time.

This chapter covers FCFS, SSTF, SCAN, C-SCAN, LOOK, C-LOOK, RAID, HDD vs SSD, and interview-focused numerical problems.
