# MCQs

## Q1

Which algorithm always gives the minimum page faults?

A. FIFO

B. LRU

C. Optimal

D. Clock

✅ Answer

Optimal

---

## Q2

Demand Paging loads pages

A. At compile time

B. At load time

C. Only when needed

D. Never

✅ Answer

Only when needed

---

## Q3

Thrashing results in

A. High CPU utilization

B. High Disk Activity

C. Low Page Faults

D. Better Performance

✅ Answer

High Disk Activity
