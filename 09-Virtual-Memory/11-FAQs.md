# 11-FAQs
# ❓ Frequently Asked Questions

## What is Virtual Memory?

A technique that uses disk as an extension of RAM.

---

## Why is Virtual Memory needed?

To execute programs larger than physical memory.

---

## What is Demand Paging?

Loading pages only when needed.

---

## What is a Page Fault?

Requested page is not present in RAM.

---

## Which algorithm gives minimum Page Faults?

Optimal Algorithm.

---

## What is Thrashing?

Continuous Page Faults causing poor performance.

---

## What is Belady's Anomaly?

Increasing frames increases Page Faults in FIFO.

---

## What is Working Set?

Collection of recently used pages.

---

## Where is Virtual Memory used?

- Windows
- Linux
- Android
- JVM
- Cloud Systems
